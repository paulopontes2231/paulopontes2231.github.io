INSTRUCTIONS

If you want to do this with the players from your game, you'll have to download this view in order to use this functionality. After you import this view on the 'Search Player' tab you're ready to start the process.

Once you have the list of players you want to rate, you'll need to press Ctrl+A on your keyboard, and when all the players are selected, you'll press Ctrl+P. After this you'll choose the option 'Web Page'. Then you save the file with a name you want and in a directory you'll have easy access to.

After saving the file you open this file with Microsoft Excel, or any similar program, then you save the file as "CSV UTF-8". This new file, is the file you'll to upload to rate the players you selected.

If you want to test this functionality and you don't want to bother yourself with this process, you can import a default list of players by clicking the button below.
The results will appear after you click the button and close this pop-up.
